Poll Management Interface

Familiarising yourself with the interface

All polls created by you will be listed in the table. Polls will be listed in descending order of recency. Descriptions help to differentiate similar/identical polls that may be meant for different audiences. Polls are controlled according based on its Status.

Status
Can be toggled on (green) or off (red).
Actions available for clicking correspond to the state of the poll.
The Status of the poll controls whether or not the poll is still open for feedback.

Actions
There are a total of three actions:

  • Get Link

    • When the Status is green, clicking on the title of the poll also directs the user to the Get Link page. Clicking on this option will display the QR Code and URL for the user to get audiences to navigate to the answering page.

  • Poll Results

    • When the Status is red, clicking on the title of the poll also directs the user to the show results page. Clicking on this option will display the results to the poll and also the export function for the user to extract the results of the polls into a csv spreadsheet.

  • Delete

    • When the Status is red, the delete option will be available for the user to click on. Deleting the poll will prompt for a confirmation from the user to delete the poll permanently.

And that's as simple as it gets!
